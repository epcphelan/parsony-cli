# Parsony CLI
## Command line tools for the Parsony Framework

### Installation
``` 
npm i -g parsony-cli 
```


Create new Parsony project

``` 
mkdir <name> 
cd <name> 
parsony init
```

Create new Service

```
parsony +s

```

Create new endpoint - if service does not yet exist, will also create new
service.

``` 
parsony +
```

Create new model

```
parsony +m
```
